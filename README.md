# Paithani Saree Website

A full-stack website showcasing authentic Paithani sarees with artisan stories, fair trade practices, and global delivery.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Testing: Postman

## Features
- Artisan-crafted sarees
- Admin dashboard and user login
- Global shipping with tracking
- Return/refund and care guide
